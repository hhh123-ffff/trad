# Release Notes - v0.1.0

发布日期：2026-03-24

## 1. 版本定位

`v0.1.0` 是项目从 0 到 1 的首个可运行基线版本，目标是交付“个人量化交易平台”的端到端闭环能力。

## 2. 主要能力

- 研究与回测：支持因子计算、策略信号、历史回测与指标分析。
- 模拟盘：支持每日信号驱动的虚拟交易、持仓与资产快照。
- 风控：支持止损、仓位限制、回撤/暴露告警与风险日志。
- 可视化：提供 Dashboard/持仓/订单/信号/回测/风控页面。
- 工程化部署：提供 Docker Compose 一键启动，包含定时任务调度。

## 3. 包含模块

- 后端：FastAPI + SQLAlchemy + pandas/numpy + APScheduler。
- 前端：Next.js + TypeScript + Ant Design + Recharts。
- 数据库：PostgreSQL（推荐）/ SQLite（开发）。
- 调度：收盘后更新数据、生成信号、执行模拟盘，周级回测任务。

## 4. 验证结果

- 后端测试：`pytest -q` 全部通过（20/20）。
- 前端构建：`npm run build` 通过。
- Docker 编排：`docker compose ... config` 校验通过。

## 5. 已知事项

- 本地环境若缺少 `apscheduler` 且网络受限，调度守护模式可能无法直接安装依赖。
- 解决方式：优先使用 Docker 的 scheduler 服务；或先用 `--once` 模式执行任务。

## 6. 升级建议（v0.2.0 方向）

- 接入真实 broker 适配器并做交易风控隔离。
- 增加策略参数寻优、批量回测与策略组合管理。
